# This repository contains my personal math self-study notes

## Structure

- `Algebra` — *Practical Algebra: A Self-Teaching Guide, 2nd Edition*
  - This is not an exhaustive or very deep text, but it gives a solid overview of basic and some intermediate algebra concepts.
  - Explanations are clear and step-by-step, suitable for beginners or anyone who needs a refresher.
  - Uses a strong *learn-by-doing* approach: numerous worked examples, frequent mini-quizzes, and self-tests provide immediate feedback.
- `Calculus` — *Quick Calculus: A Self-Teaching Guide, 3rd Edition*
  - Concise, self-paced introduction to the core ideas of calculus: limits, derivatives, integrals, and basic applications.
  - Written in a clear, informal style with step-by-step explanations aimed at readers who remember only high-school algebra.
  - Follows a structured *teach-yourself* format with many examples, and self-tests, designed to learn the basic techniques of differential and integral calculus
